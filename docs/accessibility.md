## 1. Goals
- Make core experience usable without voice
- Provide comfort options for VR motion sensitivity
- Ensure readable UI overlays in VR

## 2. Requirements
- Text chat always available
- Captions (phase 2): speech-to-text per user with consent
- Color not sole indicator (friend/stranger markers must have icons too)
- UI scale setting
- VR comfort:
  - teleport default
  - vignette option
  - snap turning option

## 3. Testing
- Screen reader support for menus (outside 3D)
- Contrast checks for overlays
- VR legibility test at typical distances
