# VirtuBar — Moderation & Operations Playbook

Version: 1.0  
Last Updated: 2026-02-06

## 1. Moderator Roles
- Admin: full access
- Moderator: can view reports, kick/ban, review evidence
- Support: can respond to tickets but cannot ban

## 2. Core Actions
- Kick from room (temporary)
- Ban from platform (timeboxed or permanent)
- Shadow mute (text only) (optional)
- Force disconnect voice

## 3. Report Triage Workflow
1) New report arrives (status=open)
2) Auto-enrichment:
   - roomId, timestamp, reported user history
3) Moderator reviews
4) Action:
   - no action
   - warning
   - kick
   - ban
5) Mark report resolved with reason

## 4. Real-time Incident Handling
If a room becomes chaotic:
- raise chat cooldown
- temporarily reduce room capacity
- enable “push-to-talk only” mode (room setting)

## 5. Trust Signals (Optional)
- Verified badge (linked account)
- Account age
- Reputation score (internal)

## 6. Safety UX Requirements
Mute/block/report accessible within 1–2 taps/clicks.
